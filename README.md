# Pattern Observer System

Sistema modulare per l’analisi comportamentale basato su:
- 30 segnali NV / PV / V
- Scale semantiche 0–3 specifiche per ogni segnale
- Matrice di congruenza
- Baseline engine
- Motore di inferenza (decision tree)
- Pattern library (25 pattern)
- Modello di intensità (0–100)
- Modello di predizione direzionale
- Interfaccia mobile per raccolta dati

## Struttura del repository

- /data → dataset dei segnali, scale, pattern
- /docs → documentazione tecnica del sistema
- /system → regole, modelli, decision tree

## Obiettivo
Fornire un sistema completo, modulare e scalabile per:
- osservazione sul campo
- analisi comportamentale
- inferenza di pattern
- predizione direzionale
