# Architettura del Pattern Observer System

Il sistema è composto da sei moduli principali:

1. Pattern Library (25 pattern, 6 cluster)
2. Matrice di Congruenza (NV–PV–V)
3. Regole di Priorità
4. Motore di Inferenza (Decision Tree)
5. Baseline Engine
6. Moduli avanzati: Intensità, Contesto, Predizione

Ogni modulo è indipendente e combinabile.
