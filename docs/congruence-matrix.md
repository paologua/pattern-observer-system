# Matrice di Congruenza

## Congruenza interna
- NV coerente → emozione stabile
- PV coerente → regolazione stabile
- V coerente → narrativa stabile

## Congruenza inter-canale
- NV = PV → emozione autentica
- NV ≠ V → dissonanza emotiva
- PV ≠ V → controllo / manipolazione
- NV ≠ PV ≠ V → conflitto interno / inganno

Regola assoluta: NV > PV > V.
